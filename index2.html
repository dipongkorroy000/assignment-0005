Question-1: What are the different ways to select an element in the DOM?

Ans: getElementById(id), getElementsByClassName(class name), getElementsByTagName(tag name), querySelector(), and querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans: "textContent" retrieves all the text contained within the element."innerText" retrieves only the visible text of an element, ignoring HTML tags. "innerHTML" retrieves the entire HTML markup contained within the element, including tags.
